# Tableau KPI Pill Extension

This project is a Tableau Extension that renders a "KPI Pill" — a compact, visual representation of a Key Performance Indicator.

## Tech Stack
- **Frontend:** HTML5, Vanilla CSS, Vanilla JavaScript.
- **Tableau API:** Uses the Tableau Extensions API (`tableau-extensions-1.latest.js`).

## Project Structure
- `index.html`: Main entry point and UI.
- `kpi-pill.trex`: Tableau Extension manifest file.
- `README.md`: General project documentation.

## Development Guidelines
- Maintain a clean, minimal UI consistent with Tableau's aesthetic.
- Ensure the extension is responsive and fits within small dashboard zones.
- Use the Tableau Extensions API for all data communication and settings.
